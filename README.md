
### Terminal Notes
Welcome to term-notes, Smart way to store your notes in terminal...

Usage: notes [-c] [-l] [-li] [-g] [-u] [-d] [-da] [-h]

| flag     | Description                  |
| -------- | ---------------------------- |
| -c       | Create a note                |
| -l       | List all notes               |
| -li      | List all notes with info     |
| -g       | Get info of note             |
| -u       | Update a note                |
| -d       | Delete a note                |
| -da      | Delete all notes             |
| -h       | Display this help message    |
