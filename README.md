
### Terminal Notes
Welcome to term-notes, Smart way to store your notes in terminal...

Usage: notes [-c] [-l] [-g] [-u] [-d] [-h]

| flag     | Description         |
| -------- | ------------------- |
| -c       | Create a note       |
| -l       | List all notes      |
| -g       | Get info of note    |
| -u       | Update a note       |
| -d       | Delete a note       |
| -h       | Display this help message   |
